1. open ubuntu
2. confirm angular version by executing node -v and npm -v
3. install angular cli by executing following command npm install -g @angular/cli
4. Next go to project folder by execute the following command cd /mnt/c/Users/Gayathri/Projects
5. To create angular project execute the following command  ng new employeemgmt
6. after angular project creation is done execute cd employeemgmt
7. execute npm install to install dependencies
8. execute the following command to start server ng serve --host 0.0.0.0 --port 4200
9. To launch vs code execute the following command code .
10. To install git execute the following command from ubuntu git init
11. git status
12. git add *
13. git commit -m"initial commit"
14. check branch git branch -M main
15. git remote add employeemgmt-ui https://github.com/chinthalapudigayathri/employeemgmt-ui
